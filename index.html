BlueCOntrol

For we all hold in our hands the power to change the future of out race.

The application developed for android platform pairs up with an arduino based hardware. The system together functions as a remote control to control home appliances like lights and fans.

The communication between the phone which acts as a remote control and the hardware which manipulates the appliances is currently carried out via bluetooth. The functionality may be extended in future to WIFI.

The code is completely open source and developers are welcome to alter it.
